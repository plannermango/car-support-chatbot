# 🚗 현대자동차 고객 상담 AI 챗봇

24시간 운영되는 자동차 긴급 상담 AI 챗봇 데모입니다.

## 주요 기능

- 🚨 긴급 상황 대응 (계기판 경고등 등)
- 💡 전문적인 자동차 지식 제공
- 📅 서비스센터 예약
- 🔧 임시 조치 방안 안내
- 📞 긴급출동 서비스 연결

## 시나리오

이 챗봇은 계기판 이상으로 인해 급하게 연락한 고객과의 실제 상담 상황을 시뮬레이션합니다.
- 긴급 상황 대응
- 교착 상황 해결
- 전문적인 기술 지식 제공
